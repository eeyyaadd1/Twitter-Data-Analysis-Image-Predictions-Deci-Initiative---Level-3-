# WeRateDogs Twitter Data Wrangling Project
### Digital Egypt Cubs Initiative (DECI) - Level 3 Project

## 📌 Project Overview
This project focuses on the **Data Wrangling** process, which is a core skill for any Data Analyst. The dataset used is the tweet archive of Twitter user `@dog_rates`, also known as **WeRateDogs**. 

The goal was to gather data from multiple sources, assess its quality and tidiness, and then clean it to perform meaningful analysis and visualizations.

## 🛠️ Skills & Tools Used
* **Python Libraries:** Pandas, NumPy, Requests, Tweepy, Matplotlib, Seaborn.
* **Data Gathering:** Manual download (CSV), Programmatic download (Requests/TSV), and JSON parsing.
* **Data Cleaning:** Handling missing values, correcting data types, merging datasets, and filtering outliers.
* **Environment:** Jupyter Notebook.

## 📂 Dataset Sources
1. **Twitter Archive Enhanced:** A CSV file containing basic tweet data for over 5,000 tweets.
2. **Image Predictions:** A TSV file hosted on Udacity's servers containing dog breed predictions generated by a Neural Network.
3. **Additional Data:** JSON data (Retweet/Favorite counts) retrieved via Twitter API.

## 🔄 The Wrangling Process
The project followed a strict three-step workflow:

### 1. Data Gathering
* Loaded `twitter_archive_enhanced.csv`.
* Downloaded `image_predictions.tsv` programmatically using the `Requests` library.
* Parsed `tweet_json.txt` to extract engagement metrics.

### 2. Data Assessment
* Identified **8 Quality Issues** (e.g., incorrect dog names like 'a' or 'an', rating numerators with decimal issues, and retweets that needed removal).
* Identified **2 Tidiness Issues** (e.g., dog stages spread across four columns instead of one).

### 3. Data Cleaning
* Used the **Define-Code-Test** framework to systematically clean the data.
* Merged the three datasets into one master DataFrame (`twitter_archive_master.csv`).

## 📊 Key Insights & Visualizations
* Analyzed the correlation between Retweets and Favorites.
* Visualized the most popular dog stages (Puppo, Doggo, Pupper, Floof).
* Identified the most common dog breeds based on Neural Network predictions.
